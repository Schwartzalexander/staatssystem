# Staatssystem Deutschland

Hier wird ein Konzept für ein neues Staatssystem für Deutschland vorgestellt. Das Repository sammelt zentrale Bausteine einer möglichen Staatsordnung und ordnet sie in thematische Bereiche.

## Inhaltsverzeichnis

- [Technokratie](#technokratie)
  - [Staatsrat](technokratie/staatsrat.md)
  - [Präsident](technokratie/praesident.md)
- [Staatsorganisation](#staatsorganisation)
  - [Föderalismus und zentrale Staatsverwaltung](staatsorganisation/foederalismus.md)
- [Sozialsystem](#sozialsystem)
  - [Krankenversicherung](sozialsystem/krankenversicherung.md)

## Technokratie

Der Bereich Technokratie beschreibt die Struktur des Staatsrats und die Rolle des Präsidenten als Staatsoberhaupt im technokratischen Modell.

## Staatsorganisation

Der Bereich Staatsorganisation behandelt den Aufbau des Staates, insbesondere die Frage von Zentralisierung und Verwaltungsgliederung.

## Sozialsystem

Der Bereich Sozialsystem beschreibt Modelle der sozialen Absicherung, aktuell mit Fokus auf die Ausgestaltung der Krankenversicherung.
