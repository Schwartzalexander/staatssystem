# Feiertage

Alle religiösen Feiertage entfallen oder werden durch weltanschaulich neutrale Feiertage ersetzt.

| Datum  | Name                | Bedeutung                                          |
|--------|---------------------|----------------------------------------------------|
| 01.01. | Neujahr             | Beginn des neuen Jahres                            |
| 01.06. | Reformtag           | Jahrestag der Staatsreform                         |
| 25.12. | Erster Familientag  | Erster Tag, an dem Familien zusammenkommen können  |
| 26.12. | Zweiter Familientag | Zweiter Tag, an dem Familien zusammenkommen können |
